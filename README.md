# Pandas Practice Notebook

A short reference notebook for core **Pandas** concepts with clear explanations and runnable examples.

## Covers
- Data loading & inspection  
- Filtering & selection  
- Column operations  
- Missing data handling  
- Sorting, aggregation & grouping  
- Merging & concatenation  

## Usage
Open `pandas_practice.ipynb`, read the notes, and run the cells.

## Requirements
Python 3.x, Pandas, Jupyter Notebook
